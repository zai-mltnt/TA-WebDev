# LeafShelf Library

LeafShelf Library is a simple web-based digital library that allows users to browse, search, and manage books. It provides an interactive interface for viewing book details, adding new books, and organizing a personal collection.

Live Demo:  
https://zaiwebdev.w3spaces.com/webdevact.html

## Features

- Browse a collection of books  
- Search books in real-time  
- Add new books  
- Delete books  
- Rating system (1–5)  
- Book categories  
- View detailed book information in a modal  
- Data persistence using Local Storage  

## Built With

- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Bootstrap 5.3  

## Project Structure

LeafShelf Library │── index.html │── README.md

## How It Works

1. The application loads with a default list of books.
2. Users can search for books using the search bar.
3. Clicking "View Book" opens a modal with detailed information.
4. Users can add new books using the form.
5. Books can be deleted from the list.
6. All data is stored in the browser's localStorage.

## How to Run Locally

1. Download or copy the code.
2. Save the file as `index.html`.
3. Open the file in any web browser.

## Notes

- No backend is required.
- Data is stored locally in the browser.
- Clearing browser data will remove saved books.
